# CollegeHub Frontend

## Overview

CollegeHub Frontend is a modern React-based user interface that allows students to explore colleges, compare institutions, save favorites, participate in discussions, and access personalized college recommendations.

The frontend focuses on providing an intuitive, responsive, and user-friendly experience across desktop and mobile devices.

---

## Features

* College listing and search
* College comparison
* Saved colleges management
* User authentication
* GitHub OAuth integration
* Discussion forum interface
* College recommendation system
* Responsive design

---

## Tech Stack

---

## Installation

Clone the repository and install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
